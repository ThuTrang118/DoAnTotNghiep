CHỨC NĂNG: TÌM KIẾM

================================================================================
MÔ TẢ
================================================================================
Chức năng cho phép người dùng tìm kiếm sản phẩm trên website.

Người dùng:
- nhập Keyword

Hệ thống kiểm tra:
- field bắt buộc
- định dạng dữ liệu
- domain sản phẩm giày/dép
- sự tồn tại của sản phẩm phù hợp

Chỉ tìm kiếm thành công khi:
- Keyword hợp lệ
- Keyword thuộc domain sản phẩm giày/dép
- Keyword tồn tại trong hệ thống

================================================================================
THÀNH PHẦN GIAO DIỆN (UI COMPONENT)
================================================================================

1. Keyword
- UI Type: Search Textbox
- Data Type: String
- Required: Yes

================================================================================
INPUT: gồm đúng 1 field
================================================================================

1. Keyword: Search Textbox, String, bắt buộc

================================================================================
CONSTRAINT
================================================================================

Keyword:
1. Bắt buộc nhập
2. Là chuỗi tìm kiếm sản phẩm
3. Phải thuộc domain sản phẩm giày/dép
4. Có thể là tên sản phẩm, một phần tên sản phẩm (Sandal Gót Vuông), mã sản phẩm (G169), màu sắc(đen) hoặc loại sản phẩm (sandal)
5. Có thể tồn tại hoặc không tồn tại trong hệ thống

VD không hợp lệ:
- điện thoại
- tủ lạnh
- máy giặt
- sandal@@@

================================================================================
QUAN HỆ DỮ LIỆU
================================================================================

Keyword thuộc domain sản phẩm giày/dép nhưng không tồn tại trong hệ thống:
→ Expected = "Không tìm thấy sản phẩm"

Keyword không thuộc domain sản phẩm giày/dép:
→ Expected = "Không tìm thấy sản phẩm"

================================================================================
KỲ VỌNG NGHIỆP VỤ
================================================================================

1. Nếu Keyword rỗng
→ Expected = "Vui lòng điền vào trường này"

2. Nếu Keyword không thuộc domain sản phẩm giày/dép
→ Expected = "Không tìm thấy sản phẩm"

3. Nếu Keyword hợp lệ nhưng không tồn tại trong hệ thống
→ Expected = "Không tìm thấy sản phẩm"

4. Nếu Keyword hợp lệ và tồn tại trong hệ thống
→ Expected = chính giá trị Keyword đã nhập
VD:
Keyword = "G169"
→ Expected = "G169"

Keyword = "sandal"
→ Expected = "sandal"
