CHỨC NĂNG: ĐĂNG NHẬP

================================================================================
MÔ TẢ
================================================================================
Chức năng đăng nhập cho phép người dùng truy cập hệ thống bằng tài khoản đã đăng ký.

Người dùng:
- nhập Username
- nhập Password

Hệ thống kiểm tra:
- field bắt buộc
- Username có tồn tại hay không
- Password có đúng với Username tương ứng hay không

Chỉ đăng nhập thành công khi:
- Username tồn tại
- Password đúng với Username tương ứng

================================================================================
THÀNH PHẦN GIAO DIỆN (UI COMPONENT)
================================================================================

1. Username
- UI Type: Textbox
- Data Type: String
- Required: Yes

2. Password
- UI Type: Password Textbox
- Data Type: String
- Required: Yes
- Dữ liệu được che ký tự

================================================================================
INPUT: gồm đúng 2 field
================================================================================

1. Username: Textbox, String, bắt buộc
2. Password: Textbox, String, bắt buộc

VD:
Username = "AnhDuong11"
Password hợp lệ = "AnhDuong@123"

================================================================================
QUAN HỆ DỮ LIỆU
================================================================================

Password phải đúng với Username tương ứng.

VD:
Username = "AnhDuong11"
Password = "AnhDuong@123"

================================================================================
KỲ VỌNG NGHIỆP VỤ
================================================================================

1. Nếu bất kỳ field nào rỗng
→ Expected = "Vui lòng điền vào trường này"

2. Nếu Username không tồn tại
→ Expected = "Tên đăng nhập hoặc mật khẩu không đúng!"

3. Nếu Username đúng nhưng Password sai
→ Expected = "Tên đăng nhập hoặc mật khẩu không đúng!"

4. Nếu:
Username = "AnhDuong11"
Password = "AnhDuong@123"

→ Expected = "AnhDuong11"
